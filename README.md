# 📌 MyRPG

Il progetto è stato realizzato per il corso di Metodologie di Programmazione, è un applicativo java che implementa un gioco di ruolo ad ambientazione fantasy.

---

## 🚀 Come eseguire il progetto

### Prerequisiti
- Java 25 (LTS)
- Git (per clonare il repository)

(Nota: Gradle è integrato nel progetto tramite il Gradle Wrapper, non è necessario installarlo localmente).

### Istruzioni

```bash
git clone https://github.com/diegoFiccadenti/RPG.git
cd RPG
```

### Build del progetto
```bash
.\gradlew build
```

### Esecuzione
```bash
.\gradlew run
```
